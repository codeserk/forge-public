# Forge

A collection of developer tools - analytics, translations, marketing assets, and more.

> Early alpha. Things will break.

## Values

**Fairness** - pricing reflects real costs. Features that don't add infrastructure cost won't affect your bill.

**Privacy** - tools are privacy-first by design. Stats does not use cookies. IPs are hashed and the hash is rotated, so they are never stored in plain form.

## Modules

- **Stats** - lightweight page view analytics
- **Texts** - translation management _(coming soon)_
- **Studio** - screenshots and marketing assets _(coming soon)_
